# clock v0.1.0

`iclock.html` は、ブラウザでローカル実行できるデジタル時計兼シーケンスタイマーです。HTML、CSS、JavaScriptを1ファイルに含めており、サーバー、npm、ビルド、CDN、外部API、外部フォント、外部ライブラリは不要です。

## 起動方法

1. リポジトリ直下の `iclock.html` を現在のChromium系ブラウザで直接開きます。
2. 音声読み上げを使う場合は、画面右上の「音声を有効化」を一度クリックします。ブラウザの自動再生制限により、初回はユーザー操作が必要です。

## 時計モード

* `HH:MM:SS` の24時間表記で時刻を表示します。
* 日付は補助情報として小さく表示します。
* 「設定」パネルで表示オフセットを分単位の整数で指定できます。

## オフセットの意味

表示オフセットは端末の実時刻に加算する分数です。

* `+10` は端末時刻より10分進めて表示します。
* `-5` は端末時刻より5分戻して表示します。
* `0` は端末の現在時刻です。

時計読み上げ判定は、端末の実時刻ではなく、このオフセット後の表示時刻を基準にします。オフセットが0でない場合は画面に「補正中」と表示されます。

## タイマーモード

* 複数シーケンスを上から順に実行します。
* 主画面は残り時間の視認性を優先し、区間名や区間番号は表示しません。
* 開始、一時停止、再開、リセット、前へ、次へを操作できます。
* Spaceで開始／一時停止／再開、Rでリセット、Escapeで設定パネルを閉じられます（入力欄編集中を除く）。
* 残り時間は通常 `MM:SS`、1時間以上では `HH:MM:SS` で表示します。負数表示にはしません。

## シーケンス

設定パネルでシーケンスの追加、名前編集、秒数編集、削除、上方向への並べ替えができます。秒数が1以上で、名前が空でないシーケンスだけが実行対象です。初期値は短時間で確認しやすい8秒、6秒、4秒の3区間です。

## 読み上げ設定

ブラウザ標準の Speech Synthesis API を使用します。外部音声サービスや外部通信は使いません。音声の種類、再生可否、発音品質はブラウザやOS環境に依存します。

* 時計読み上げは最大5件です。有効、時、分、秒、文章を設定できます。同じ日の同じ条件は1回だけ発火します。設定変更、日付変更、オフセット変更、読み上げ状態リセットで再判定できます。
* タイマー読み上げは最大5件で、全シーケンス共通です。有効、残り秒数、文章を設定できます。同一シーケンス内では同じ条件を1回だけ発火し、次シーケンスへ進むと発火済み状態をリセットします。
* 読み上げキューが無制限に蓄積しないよう、アプリが発話する直前に `speechSynthesis.cancel()` を呼んでから `speak()` します。タイマー計測は読み上げ完了を待たず継続します。

## 時間管理方式

タイマーは `setInterval` の呼び出し回数を数えて減算しません。各シーケンス開始・再開時の `performance.now()` と、一時停止までの経過秒数から毎回残り時間を計算します。画面更新間隔と時間計算は分離しているため、描画遅延や非アクティブタブからの復帰時も累積ずれを抑えます。

読み上げ閾値は、前回残り時間と今回残り時間の間で指定秒数を通過したかで判定します。0秒の読み上げは先に発火済みにしてから、短い非同期遅延後に次シーケンスへ移動します。これにより、0秒メッセージの取りこぼしとシーケンス切替時の端数持ち越しを防ぎます。

## テーマと保存

ライト／ダークテーマを手動切替できます。設定は `localStorage` にバージョン番号付きJSONとして保存されます。保存対象は、最後のモード、テーマ、時計オフセット、時計読み上げ、タイマー読み上げ、シーケンス、音声速度です。壊れた保存値がある場合は初期値に戻して起動します。「設定を初期化」で保存内容を初期化できます。

## 外部通信

このツールは外部通信を行いません。個人情報や利用情報を外部送信しません。

## 動作確認環境

* Chromium系ブラウザを主対象に実装しています。
* 開発時はローカルファイルとして開ける構成、外部依存がない構成、JavaScript構文エラーがないことを確認しています。

## 既知の制約

* 音声読み上げの実際の声、音量、対応言語、初回再生可否はブラウザとOSの Speech Synthesis 実装に依存します。
* 実行中タイマー状態の自動復元は行いません。ページ再読込後は停止状態から開始します。
* 初期実装のタイマー読み上げ設定は全シーケンス共通です。
